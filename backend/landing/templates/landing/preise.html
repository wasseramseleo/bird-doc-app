{% blocktranslate trimmed %}Was kostet BirdDoc?{% endblocktranslate %}

{% blocktranslate trimmed %}BirdDoc wird pro Organisation lizenziert, nie pro Kopf: Beringer ohne eigenes Konto erfassen kostenlos mit, und Organisationen der Beta-Kohorte behalten dauerhaft ihren Vorzugspreis – das ist das Preismodell, unabhängig vom späteren Betrag.{% endblocktranslate %}

{% include "landing/_app_german_only_note.html" %}

  • {% translate "Pro Organisation, nie pro Kopf" %}

    {% blocktranslate trimmed %}Die Lizenz gilt pro Organisation, nicht pro Person. Eine Station oder ein Verein zahlt einmal für die ganze Gruppe, statt jede beringende Person einzeln zu lizenzieren – der Preis wächst nicht mit jedem zusätzlichen Beringer.{% endblocktranslate %}

  • {% translate "Beringer ohne Konto kostenlos" %}

    {% blocktranslate trimmed %}Beringer ohne eigenes Konto erfassen ihre Fänge kostenlos mit. Wer nur gelegentlich an der Station beringt, belegt keinen kostenpflichtigen Platz und verursacht keine zusätzlichen Kosten.{% endblocktranslate %}

  • {% translate "Beta-Kohorte behält den Vorzugspreis" %}

    {% blocktranslate trimmed %}Organisationen, die während der öffentlichen Beta einsteigen, gehören zur Beta-Kohorte und behalten dauerhaft ihren Vorzugspreis. Ein späterer Regelpreis gilt für sie nicht, auch wenn die Beta endet.{% endblocktranslate %}

{% comment %} Der aktuelle Beta-Status wird ehrlich genannt (issue #304), aber nicht zum zitierbaren Kern gemacht: der Kern ist das Modell oben, nicht der Betrag. {% endcomment %}
