{% blocktranslate trimmed %}Funktionen einer Beringungssoftware{% endblocktranslate %}

{% blocktranslate trimmed %}Eine Beringungssoftware sollte am Ringtisch offline erfassen, Fänge als IWM-Datei an die Vogelwarte exportieren, unplausible Eingaben sofort als Warnung anzeigen und Ringnummern fortlaufend aus der Ringserie vorschlagen – das sind die vier Kernfunktionen, die BirdDoc erfüllt.{% endblocktranslate %}

{% include "landing/_app_german_only_note.html" %}

  • {% translate "Offline-Fähigkeit" %}

    {% blocktranslate trimmed %}Offline-Fähigkeit heißt, dass jeder Fang direkt am Ringtisch erfasst wird, auch ganz ohne Internet. Die Daten liegen sicher auf dem Gerät und werden abgeglichen, sobald wieder eine Verbindung besteht.{% endblocktranslate %}

  • {% translate "IWM-Export" %}

    {% blocktranslate trimmed %}Der IWM-Export erzeugt aus allen Fängen eines Projekts eine fertige Meldedatei im IWM-Format der Vogelwarte – ohne die Tabelle von Hand umzubauen.{% endblocktranslate %}

  • {% translate "Plausibilitätswarnung" %}

    {% blocktranslate trimmed %}Eine Plausibilitätswarnung zeigt schon beim Erfassen an, wenn ein Maß oder eine Angabe für die gewählte Art außerhalb des üblichen Bereichs liegt – der Fehler fällt am Ringtisch auf, nicht erst Monate später.{% endblocktranslate %}

  • {% translate "Ringserien-Logik" %}

    {% blocktranslate trimmed %}Die Ringserien-Logik führt jede Ringserie fortlaufend und schlägt die nächste freie Ringnummer automatisch vor – Zahlendreher und doppelt vergebene Ringe werden vermieden.{% endblocktranslate %}
