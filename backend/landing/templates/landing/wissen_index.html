Wissen — die öffentliche Beringungs-Referenz

Die Wissen-Referenz von BirdDoc erklärt und belegt die Grundlagen der Vogelberingung in Österreich — offen, ohne Anmeldung und in klarer Sprache. Sie besteht aus drei Sektionen: der Ringgrößen-Tabelle, den Artenseiten und dem Beringungs-Glossar.

  • Ringgrößen-Tabelle

    Die vollständige Tabelle nennt die Empfohlene Ringgröße für jede Art der österreichischen Artenliste — mit deutschem Namen, wissenschaftlichem Namen, Familie und Ordnung.

  • {% if example_art_url %} Artenseiten {% else %} Artenseiten {% endif %}

    Eine ausführliche Seite je Art mit ihrer Empfohlenen Ringgröße und Taxonomie. Alle Artenseiten sind über jede Zeile der Ringgrößen-Tabelle erreichbar{% if example_species %} — zum Beispiel die Seite zur {{ example_species.common_name_de }}{% endif %}.

  • Beringungs-Glossar

    Das Glossar erklärt die Fachbegriffe der Vogelberingung — von Erstfang über Wiederfang bis Ringserie, je ein Begriff auf einer eigenen Seite, antwortzuerst.
