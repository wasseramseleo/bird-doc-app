# CLAUDE.md

Project instructions for agents working in this repo.

## Agent skills

### Issue tracker

Issues and PRDs live in this repo's GitHub Issues (`wasseramseleo/bird-doc-app`), managed via the `gh` CLI. See `docs/agents/issue-tracker.md`.

### Triage labels

Canonical triage roles map to identically-named labels (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
